# JACOBI

## Instalacion
- Accede al directorio del proyecto.

```bash
cd LU
```

- Crear un entorno virtual. (opcional)

```bash
python3 -m venv venv
source venv/bin/activate
```

- Instala las dependencias.

```bash
pip3 install -r requirements.txt
```

- Ejecuta el programa.

```bash
python3 lu.py
```

- Sigue las instrucciones que aparecen en consola.
